# README

## Tema yang Dipilih
Tema yang dipilih untuk website ini adalah **Portofolio Profesional Teknologi**. Website ini dirancang untuk menampilkan keahlian dan layanan yang ditawarkan, terutama di bidang Internet of Things (IoT), Artificial Intelligence (AI), dan Computer Vision.

## Fitur-Fitur Utama
1. **Halaman Utama**:
   - Menampilkan informasi pribadi pemilik website, termasuk bidang minat dan keahlian.
   - Navigasi yang mudah digunakan untuk membantu pengguna mengeksplorasi layanan dan proyek.

2. **Bagian Layanan Vertikal**:
   - Layanan seperti IoT Engineering, Artificial Intelligence, dan Computer Vision disusun secara vertikal.
   - Setiap layanan dilengkapi dengan ikon yang relevan, judul yang jelas, dan deskripsi singkat.

3. **Proyek Portofolio**:
   - Menampilkan proyek-proyek unggulan yang melibatkan IoT, pengembangan Android, AI, dan desain 3D.
   - Penjelasan detail mengenai implementasi teknologi pada setiap proyek.

4. **Tautan Media Sosial**:
   - Tautan ke profil media sosial (GitHub, LinkedIn, Instagram) untuk memperluas koneksi profesional.

5. **Desain Responsif**:
   - Website dirancang agar tampil optimal di perangkat desktop maupun seluler.

6. **Navigasi yang Rapi**:
   - Menu dan tombol navigasi sederhana untuk memberikan pengalaman pengguna yang mulus.

7. **Contact**:
   -Kontak saya melalui form dengan mengisi nama, email, dan deskripsi.

## Teknologi yang Digunakan
1. **HTML dan CSS**:
   - Digunakan untuk membangun struktur dan desain tampilan website.

2. **JavaScript**:
   - Menambahkan interaktivitas sederhana untuk animasi atau efek visual pada elemen layanan.

3. **Ikon SVG**:
   - Semua ikon layanan menggunakan SVG untuk memastikan kualitas visual yang tajam di berbagai resolusi layar.

4. **Hosting Domain**:
   - Website ini dihosting menggunakan nama domain `www.pixelmind.my.id`.

4. **Formspree**:
   - pesan pengirim pada contact yang dikirimkan ke formspree akan diteruskan ke email saya.

## Cara Menjalankan Website
1. Pastikan Anda memiliki koneksi internet untuk mengakses website di [PixelMind](https://www.pixelmind.my.id/).
2. Navigasi ke bagian layanan untuk melihat keahlian yang ditawarkan.
3. Klik pada tautan media sosial untuk terhubung dengan profil profesional pemilik website.
